Sistema de Irrigação

Projeto web desenvolvido com Python (Flask) que permite o monitoramento em tempo real da situação de válvulas de irrigação em hortas ou jardins automatizados. A aplicação exibe o estado atual de cada válvula (aberta ou fechada), facilitando a gestão remota da irrigação. Além disso, integra uma API de clima para mostrar dados essenciais como temperatura, umidade, vento e outras condições ambientais, ajudando a tomar decisões mais precisas sobre a irrigação. Ideal para agricultura de precisão e controle automatizado de pequenos cultivos.

API Barbearia

API RESTful desenvolvida para gerenciar o fluxo de agendamentos de horários em uma barbearia. Permite o cadastro de clientes, controle de serviços oferecidos, horários disponíveis, agendamentos e valores de cada atendimento. A solução visa facilitar a organização do atendimento, evitando conflitos de horário e automatizando o controle financeiro dos serviços prestados.
